## API
1. alert: 消息确认框
```
	`|` alert(')
```

2. confirm: 对话框

3. toast: 提醒信息

4. getIMEI: 获取设备Id

5. getUA: 获取

6. finish: 完成，页面生命周期结束，视图实例被注销

7. swipeView: 视图切换(h5-to-native)

8. getNetworkType: 获取网络类型
