# about js
## Usage
* install node
* install gulp global
* npm install
* gulp
* http://localhost:8093/test/test.html

## Description
hybrid app 前端工具
* 工具涵盖了android版本和ios版本
* 客户端请在本地载入js文件, 'build>index.bundle.min.js', 如果无法在本地载入该文件, 请通过script标签载入
* 所有方法都是同步调用方式
